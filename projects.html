Simple Processor

Languages Used: SystemVerilog

Tools Used: Git

The Simple Processor was created over a series of seven lab assignments in my first term as a computer engineering student. The course was CPEN 211 - Computing Systems I, and the labs themselves built up towards a simple processor using SystemVerilog. From this project, I was able to create a processor that incorperated model RAM - which stored instructions and data within a .txt file, model CPU - which had a decoder, controller, and datapath, and a model ALU - which did simple arithmetic. Together, the simple processor was able to read instructions from another .txt file. From this project, I was able to learn how to integrate multiple modules using SystemVerilog, and how delicate the process of connecting parts is.

Introduction to Android Development Applicaiton

Languages Used: Java, XML

Frameworks Used: Express, Google Maps API

Tools Used: Git, Gradle

This project was the first deliverable of my CPEN 321 - (Advanced) Software Engineering course. In a span of two weeks, I was able to learn and create an android application that encapsulated the workings of android application development. I learned how to set up an AWS server and through using the Express framework, I established communication between a device and the my server. Furthermore, I had implement my own functionality: a QR code scanner that scans an uploaded image for QR codes and displays the result. As there is no such functionality built into either android or apple, instead of having to share a screenshot of the QR code to a different device before scanning the code, my application would take the screenshot and return it's contents.
