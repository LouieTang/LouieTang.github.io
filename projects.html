Simple Processor


Description: An escalation of six labs/assignments. Started from datapaths, and logic, to decoders, to ALUs, to a functioning simple processor.

Language: System Verilog

Version Control: Git, GitHub

Course: CPEN 211 - Computing Systems I


Highlights: The Simple Processor was created over a series of seven lab assignments in my first term as a computer engineering student. The course was CPEN 211 - Computing Systems I, and the labs themselves built up towards a simple processor using System Verilog. From this project, I was able to create a processor that incorperated model RAM - which stored instructions and data within a .txt file, model CPU - which had a decoder, controller, and datapath, and a model ALU - which did simple arithmetic. Together, the simple processor was able to read instructions from another .txt file. From this project, I was able to learn how to integrate multiple modules using System Verilog, and how delicate the process of connecting parts is.
